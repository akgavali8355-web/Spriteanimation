# 🎬 Sprite Animation Website

A modern sprite animation website built using HTML, CSS, JavaScript, and Vite.

## 🚀 Features

- Smooth sprite animations
- Responsive design
- Fast performance
- Modern UI
- Easy to customize

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Vite

## 📂 Project Structure

```
├── public/
├── src/
├── index.html
├── package.json
└── README.md
```

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/akgavali8355-web/Spriteanimation.git
```

Go to the project folder:

```bash
cd Spriteanimation
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
